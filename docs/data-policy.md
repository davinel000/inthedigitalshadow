# Data Policy

This repository contains the public layer of the project.

## Public Material

- thesis text and short methodological notes;
- depersonalized or author-controlled excerpts from research traces;
- derived summaries, counts, timelines, and visualizations;
- final or near-final technical snapshots needed to understand system logic;
- short descriptions of tools, scripts, and analysis steps.

## Local Material

- full private chat corpora;
- unfiltered third-party messages;
- raw health or self-tracking exports that remain personally sensitive;
- local LLM-analysis inputs that contain restricted material;
- redundant historical versions, temporary folders, backups, installers, and caches;
- working environments whose publication would expose noise without adding interpretive value.

## Delegate-Analysis Boundary

`delegate-analysis` supplies selected public outputs to this repository:

- dataset schema;
- pipeline overview;
- script inventory;
- privacy-safe tables;
- final figures;
- short findings summaries.

The full local repository is not reproduced here.

## Local Filtering And Analysis

Some analytical steps remain local:

- filtering of third-party messages was performed locally;
- privacy-sensitive corpora were reduced before entering any public workflow;
- some LLM-assisted classification or extraction steps were run only on local safe subsets or author-controlled messages.

Public outputs document the results of those steps without exposing raw unsafe inputs.

## AI Assistance Note

This project used AI systems as editorial, analytical, and technical assistants at selected stages. This includes help with code, restructuring, drafting, and limited analysis support. These systems did not replace authorial responsibility.
