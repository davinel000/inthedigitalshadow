# Templates

Minimal reusable table structures based on the analytical datasets used in the thesis.

Current templates:

- `project_mass_daily_template.csv`
  - daily accumulation structure used for project mass views
- `allocate_budget_weekly_template.csv`
  - weekly budget totals grouped by spending category
- `allocate_timeline_dataset_template.csv`
  - timeline blocks used for allocation views
- `allocate_event_markers_template.csv`
  - dated event markers for overlays and annotations

Use these as starting structures for your own material. Keep raw source data local if needed, then derive tables in the same shape.
